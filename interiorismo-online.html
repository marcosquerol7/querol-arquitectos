Plan light

El servicio del plan light es para aquellos clientes que tienen mobiliario y algunos elementos para decorar, pero quieren un pequeño cambio en esa estancia. Se proporcionarán recursos para poder aprovechar los muebles que ya tiene el cliente y darle una nueva visión al espacio requerido. Buscaremos referencias, elementos decorativos, una nueva o adaptada gama cromática y nuevas texturas para la estancia de la casa, sin necesidad de cambiar el mobiliario por completo. Es un plan que puede ser más típico para estancias como dormitorios o comedores, aunque puede adaptarse a cualquier estancia.

Plan medium

El plan medium es un servicio donde aprovechamos lo máximo posible el mobiliario que cuenta el cliente pero consiguiendo un nuevo espacio. Para ello, hablaremos con el cliente para tener una serie de referencias sobre el estilo que se quiere conseguir, se proporcionarán ideas y el listado de un nuevo mobiliario y decoración. También se proporcionarán junto a las referencias documentación gráfica sobre las nuevas texturas y gama de colores. Nos basamos en un presupuesto que nos indicará el cliente para conseguir una nueva estancia.

Contacta con nosotros para tener más información.
Desde 200€

Plan full

Cambiamos tu estancia al completo. Si lo que tienes es una estancia vacía o quieres renovarla y cambiar tu estilo y mobiliario, este es tu servicio. Acordaremos un presupuesto contigo y diseñaremos tu estancia desde 0 con nuevo mobiliario, nuevo estilo, nuevas tonalidades, es decir, un nuevo espacio. Este servicio es el plan más completo, y es totalmente adaptable para cualquier tipo de estancia de la casa. Sin embargo, especialmente es un plan idóneo para la reforma integral de cocina o de baños, ya que son los espacios que normalmente se suelen necesitar una reforma integral.

Contacta con nosotros para tener más información.
Desde 300€
